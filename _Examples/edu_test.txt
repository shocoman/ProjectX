// Схема трансляции для оператора if позволяет избежать дублирования некоторых переходов.
// (на практике могут потребоваться дополнительные нетерминалы и продукции).
// Для компактности мы ввели действия непосредственно в продукцию
//  Синтаксис:
//  if (expression) statement1 else statement2
//  Действия в продукциях:
//  if(expression) {C1} statement1 {C2} else statement2 {C3}
// Действиями являются:
//  - C1 – увеличить номер метки; образовать код для перехода к метке,
// если expression ложно; поместить номер метки в стек.
//  - C2 – увеличить номер метки; образовать код для безусловного перехода к метке;
// извлечь из стека метку Lk; установить метку Lk в коде; поместить в стек метку,
// применявшуюся в безусловном переходе.
//  - C3 – извлечь из стека метку Lj; установить метку Lj в коде.
// При использовании данной грамматики будет создан следующий код:
// Пример:
$t0 = a < 5 
expression = $t0 
// Код для вычисления expression
$t1 = expression  
ifFalse $t1 goto $L1 
b = 1  
// код для statement1
goto $L2
$L1: b = 2 
// код для statement2
$L2: halt
