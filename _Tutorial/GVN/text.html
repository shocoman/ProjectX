Global Value Numbering (Нумерация значений)

За подробной информацией по этому алгоритму рекомендуем обратиться к книге "Engineering a Compiler. 2nd Edition"

Нумерация значений - один из видов анализа потока данных, применяемый оптимизирующим компилятором с целью обнаружения избыточных вычислений в коде программы. Например, "a = 1 + 1; b = 1 + 1" будет заменено на "a = 1 + 1; b = a", поскольку и 'a', и 'b' вычисляют одно и то же выражение "1 + 1".

Алгоритм глобальной нумерации значение требует, чтобы программа находилась в SSA-форме. Во время работы алгоритм осуществляет обход по дереву доминаторов. Для каждого блока поддерживается своя таблица значений. Также блоки получают доступ к таблице значений родителей в дереве доминаторов.
